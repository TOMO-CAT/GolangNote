if-else条件语句

条件语句写法

1. 普通条件语句

if condition1 {
    // ...
} else if condition2 {
    // ...
} else if condition3 {
    // ...
} else {
    // ...
}

2. 添加变量初始化语句的条件语句

if x := 20; x > 19 {
    // ...
}

注意事项

  • 注意无论语句体内有几条语句，花括号{}都是必须的

  • 左花括号{必须与if或者else处于同一行

  • Go是强类型语言，条件表达式必须严格返回布尔型数据（nil、0和1都不行）
